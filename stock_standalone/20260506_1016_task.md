# 20260506_1016 任务清单 - 赛马面板详情子窗口整理布局与三行高度优化

## 核心任务目标
优化竞价赛马面板的详情子窗口整理函数 `_arrange_detail_windows`，实现：
1. **三行数据展示**：将整理时的默认高度限制 `min_h` 调整为 `180px`，使得整理后的子窗口外观极为精悍，默认恰好完整展示 3 行个股数据。
2. **多列智能换行（防止底部重叠）**：在垂直联排排列时，如遇子窗口超出屏幕/主窗底部边界，自动执行换列换行机制，将子窗口依次向右移动一列排列，而非在底部完全重叠。

## 实施详情
- [x] **高度极致压缩**：在 `_arrange_detail_windows` 中将 `min_h` 精简至 `180` 像素。
- [x] **智能多列换列换行**：
  - 引入 `curr_y + target_h > limit_y_bottom + 10` 临界点判定。
  - 触发溢出时，自动使 `col_x` 向右移动当前窗口宽带加 `2px` 间隙（`col_x += target_w + 2`），并将 `curr_y` 重置回起点 `col_base_y`，开启新的一列排列。
  - 完美保障所有子窗口即使极其繁杂，也能以阶梯式的非重叠多列矩阵顺滑渲染。

## 完成状态

- [x] 完成子窗口防底部重叠的往右多列换行磁吸排列
