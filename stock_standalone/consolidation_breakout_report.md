## 4. “企稳加速起爆”策略在 pyQuant3 中的落地方法

交易员可以使用此策略在早盘 **09:35 - 10:00** 之间快速发现即将或者刚刚开启加速的黑马：

### 4.1 核心策略算法
在策略库（如 `stock_live_strategy.py`）中实现以下条件过滤方法：

```python
def check_consolidation_breakout(df_all):
    # 1. 过滤：最近几日反弹企稳后今日加速模型
    df_breakout = df_all[
        (df_all['lastdu4'] >= 10) & (df_all['lastdu4'] <= 40) &               # 企稳整理了 10~40 天
        (df_all['open'] / df_all['lastp1d'] >= 0.970) & (df_all['open'] / df_all['lastp1d'] <= 1.012) & # 开盘在平稳区间
        (df_all['high'] > df_all['high4']) &                                  # 今日最高价突破4日最高价
        (df_all['dff2'] >= 20.0) & (df_all['dff3'] >= 90.0) &                 # 主力大单暴量流入
        (df_all['percent'] >= 1.0) & (df_all['percent'] <= 5.5)               # 介入涨幅区间（刚红盘起步，不过度追高）
    ]
    return df_breakout
```

### 4.2 极佳买点介入时机 (Best Entry Timing)
* **第一买点 (早盘支撑位)**：09:30 - 09:40 之间，若个股在 `-1.0%` 到 `1.0%` 附近，向下测试 5 日均线不破（即 `low_to_ma5 <= 2.5%`），同时 `dff2` 率先异动过 15。此时为**分时回踩低吸点**。
* **第二买点 (突破加速点)**：09:35 - 10:15 之间，股价开始往上拉升，分时图拉出大阳线，瞬间突破前 4 日高点（`high > high4`），并且成交量比 `vol_ratio` 明显大于 1.0，`dff2` 突破 20。此时为**突破加速追涨点**。
