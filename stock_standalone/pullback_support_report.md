## 2. 系统核心指标 dff2/dff3 的底层算法与物理含义对齐

根据系统底层文件 `JSONData\tdx_data_Day.py` 的 `get_tdx_exp_low_or_high_power` 提取机制，我们可以对 `llow` 与 `minclose` 的算法逻辑以及由此衍生的 `dff2/dff3` 进行精准还原：

### 2.1 llow（最近结构低点）与 minclose（区间历史绝对低点）的提取逻辑
1. **`llow`（最近一次回踩10日均线区间的结构最低价）**：
   * 在日线周期下，系统自动定位最近一次**最低价跌破 10日均线（`low < ma10d`）**的交易日位置，并从该破位点往前寻找**“最近回踩结构中的最低价（`low.idxmin()`）”**。
   * 👉 **物理意义**：代表**个股上一波段回调企稳的结构前低（重要的前低支撑位）**。
2. **`minclose`（区间历史最低收盘价）**：
   * 系统在整段历史数据（`dz` 区间，通常为 `lastdays * 3` 也就是几百天的大区间）中，寻找**“绝对历史最低收盘价（`close.idxmin()`）”**。
   * 👉 **物理意义**：代表**个股的长周期历史大底（本轮大级别行情的绝对地平线）**。

### 2.2 dff2 与 dff3 派生指标的真实物理坐标
结合 `data_utils.py` 里的计算公式：
* **`dff2`** = `((buy - llow) / llow * 100).round(1)`  
  👉 **物理意义**：当前股价偏离**上一次波段整理底部支撑位 (`llow`)** 的涨幅百分比。
  * **强势股良性回调**（如海光信息）在拉升一波后回调踩支撑时，由于其脱离了上一个整理区间，`dff2` 通常在 **`15% ~ 25%`** 左右，代表拉升已有空间但回调踩稳。
  * **破位下行个股**（如中科电气、大连重工）由于股价创新低或运行在历史前低之下，其 `dff2` 极低（甚至是负数），表征其仍在向下泥潭中探底。
* **`dff3`** = `((buy - minclose) / minclose * 100).round(1)`  
  👉 **物理意义**：当前股价偏离**历史大底 (`minclose`)** 的上涨幅度（表征个股大周期/波段的主力做多涨幅与行情纵深）。
  * 海光信息 `dff3` 达 **60.4%**，代表大趋势完全走牛；而中科电气仅 8.7%，大连重工为负，属于大周期无资金护盘的弱势结构。

## 4. 重点战法：中京电子式强庄二次起爆（尾盘低吸与窄止损）策略

用户提供的 **中京电子 (002579)** 是极具代表性的强庄二次起爆模板：
个股经历了前期大主升浪（拉升至 21.80）后进行了深幅缩量回调（洗盘至 14.61 附近），踩稳 **20日生命线 (MA20d)** 和 **60日决策线 (MA60d)** 附近，形成多日横盘整理。

### 4.1 中京电子式“T-1日/T-2日 尾盘潜伏”条件
对于长期跟踪的重点个股与重点板块成份股，当满足以下特征时，**尾盘 (14:50 - 15:00) 是最从容、确定性最高的介入时点（如前天 6月16日 尾盘）**：
1. **牛市大格局**：`ma20d > ma60d`，且价格稳居 `ma60d` 之上（表明强庄主升趋势未被破坏）。
2. **大周期涨幅明显**：`dff3 >= 20.0%`（代表前期主力拉升纵深足够，回调后有二次启动潜力）。
3. **均线群多重支撑**：股价连续回调数日后，实体或下影线回踩在 `ma20d` 或 `ma8d`/`ma5d` 支撑线上（偏离度在 -1.5% 到 1.5% 之间）。
4. **地量洗盘完毕**：成交量相比前几日大幅萎缩，K线收缩量十字星、锤子线或小星阳线，代表短线浮筹出局，抛压衰竭。

### 4.2 超窄防守离场硬约束（坚决不抱幻想）
低吸建仓后，将盈亏比拉到极致（极窄的止损博取极大的主升加速）：
* **离场硬条件1**：盘中股价跌破**前一日的最低价 (`low < llow_prev1d`)**，说明重心下移，支撑失效。
* **离场硬条件2**：收盘跌破 **5日生命线 (`close < ma5d`)**。
> 👉 **执行原则**：一旦触发上述任一条件，必须果断割肉离场。此方法将单笔交易的潜在亏损强行锁死在 **`1.5% ~ 2.5%`** 以内，确保本金绝对安全。

### 4.3 T日加速起爆捕获（右侧主升发令枪）
若前天尾盘未入，T日（昨天 6月17日）早盘 **09:30 - 09:45** 为加速确认介入点：
* 早盘开盘平稳，成交量比 `vol_ratio` 相比前几日大幅放大（`vol_ratio >= 1.2`）。
* 分时图拉出持续性大阳线，瞬间突破昨日最高价（`high > lasth1d`），且股价迅速拉离分时均价线（VWAP）呈单边多头加速状态。

## 6. T+1 日内分时风控：分时均线（VWAP）离场机制

当我们在 T-1 日低吸潜伏或在 T 日介入后，对于冲高回落的防守，系统必须提供硬性的分时风控指标：

### 6.1 分时破位离场规则
1. **破均价线（VWAP）离场**：
   分时均价线（当日成交总额 / 当日成交总量）是市场全天的持仓成本线。**一旦分时股价由上向下击穿均价线，且在 5 分钟内无法收回，属于铁律级别的离场信号**。
2. **回落幅度警戒线**：
   日内最大涨幅若超过 8%，一旦自日内最高点回落幅度达到 **`3.5% ~ 5.0%`**，即使未跌配均价线，也应先平仓锁定利润（落袋为安），防止宽幅震荡洗盘变成趋势杀跌。
