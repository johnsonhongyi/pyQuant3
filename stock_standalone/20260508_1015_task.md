# 🏁 任务清单 & 实施记录 (2026-05-08 10:15)

## 📌 任务背景
在查看策略选股历史记录（复盘历史）时，用户发现个股当日的 `Rank` 等数据无法加载，部分没有被持久化存盘。本任务旨在深度剖析选股列表中所有未持久化的字段属性，并通过升级数据库表结构与自愈迁移机制，从根本上解决历史选股完美回溯的痛点问题。

---

## 🎯 本期完成目标
1. [x] **未持久化字段深度剖析**：对比选股界面（Treeview）的展现列与 SQLite 表结构设计（`selection_history`），分析归纳所有未被持久化落库的字段。
2. [x] **实现 SQLite 自动、无损自愈迁移**：在 `trading_logger.py` 中扩展 `selection_history` 的表结构定义，通过 `check_cols` 池使数据库在每次主程序启动时能够自动、无损、增量式地为用户补充这些缺失列，避免破坏历史已存数据。
3. [x] **升级 `log_selection` 批量写入映射**：升级 SQL `INSERT OR REPLACE` 语句，使日内最新的 `rank`、`yesterday_pct`、`sum_perc`、`win`、`open`、`stage`、`user_status`、`user_reason` 等能随策略选股一并落入本地数据库。
4. [x] **升级 `StockSelector` 的存盘数据结构**：让产生的 `record` 数据在当天产生时即完整保留上述高频属性。
5. [x] **实现历史加载的别名兼渲染别名转换**：在 `StockSelectionWindow.load_data()` 装载历史数据阶段，自动为历史载入的 `yesterday_pct`、`sum_perc` 和 `rank` 转换成 UI 显示认识的别名，实现 100% 完美不留遗憾的选股历史精确回溯！

## 🛠 核心代码修改记录

### 1. 数据库升级与字段自动补足 (`trading_logger.py`)
- 在 `selection_history` CREATE 语句中，补足以下列：
  ```sqlite
  rank INTEGER DEFAULT 0,
  zhuli_rank TEXT DEFAULT '-',
  yesterday_pct REAL DEFAULT 0.0,
  sum_perc REAL DEFAULT 0.0,
  win REAL DEFAULT 0.0,
  open REAL DEFAULT 0.0,
  stage INTEGER DEFAULT 2,
  user_status TEXT DEFAULT '待定',
  user_reason TEXT DEFAULT ''
  ```
- 将新列补充到 `check_cols` 中，由 `_init_db` 的字段迁移机制自启动自检补足。
- 升级 SQL `INSERT OR REPLACE INTO selection_history` 语句。

### 2. 选股策略生成存盘结构映射 (`stock_selector.py`)
- 在 `filter_strong_stocks` 决策后的 `record` 中追加对应属性提取：
  ```python
  'rank': int(data.get('Rank', data.get('rank', 0))),
  'yesterday_pct': float(data.get('per1d', 0.0)),
  'sum_perc': float(data.get('sum_perc', 0.0)),
  'win': float(data.get('win', 0.0)),
  'user_status': '待定',
  'user_reason': ''
  ```

### 3. GUI 加载历史记录别名转换 (`stock_selection_window.py`)
- 历史模式下，逻辑日期变更，在 `load_data()` 数据载入后做 rename 列映射，打通渲染层：
  ```python
  rename_map = {}
  if 'yesterday_pct' in self.df_full_candidates.columns:
      rename_map['yesterday_pct'] = '昨日涨幅'
  if 'sum_perc' in self.df_full_candidates.columns:
      rename_map['sum_perc'] = '连阳涨幅'
  if 'rank' in self.df_full_candidates.columns:
      rename_map['rank'] = 'Rank'
  
  if rename_map:
      self.df_full_candidates.rename(columns=rename_map, inplace=True)
  ```
